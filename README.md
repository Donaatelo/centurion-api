# centurion-api

Centurion API. Developed with node.js & express

## System requirement

. MongoDb <br/>
. Node.js <br/>
. npm

## Dependencies

. bcryptjs : 2.4.3 <br/>
. body-parser : 1.16.1 <br/>
. express : 4.14.1 <br/>
. jsonwebtoken : 7.4.1 <br/>
. mongoose : 4.8.5

## Install

. install system requierements <br/>
. clone this depot <br/>
. run command : npm install <br/>
. start api with: npm start
