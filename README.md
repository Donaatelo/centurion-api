# centurion-api

Centurion API. Developed with node.js & express

## System requirement

. MongoDb
. Node.js
. npm

## Dependencies

. bcryptjs : 2.4.3
. body-parser : 1.16.1
. express : 4.14.1
. jsonwebtoken : 7.4.1
. mongoose : 4.8.5

## Install

. install system requierements
. clone this depot
. run command : npm install
. start api with: npm start
